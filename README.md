# PlotArchive

Lokales Dokumenten-Management-System für Plotdateien.

Verwalte SVG, DXF, JPG, PNG, CWPRJ und weitere Formate übersichtlich auf deinem PC – mit Vorschaubildern, Tags, Kategorien, Projekten und Volltextsuche.

## Download & Installation

1. Unter [Releases](../../releases/latest) die neueste `PlotArchive Setup x.x.x.exe` herunterladen
2. Installer ausführen
3. PlotArchive starten

> **Hinweis:** Windows zeigt beim ersten Start möglicherweise eine SmartScreen-Warnung.  
> → "Weitere Informationen" → "Trotzdem ausführen" klicken.

## Unterstützte Dateiformate

SVG · DXF · JPG · PNG · GIF · WebP · BMP · TIFF · CWPRJ

## Features

- Ordner scannen und automatisch indizieren
- Vorschaubilder für alle gängigen Bildformate
- Tags, Kategorien und Projekte zur Organisation
- Designer aus Datei-Metadaten automatisch erkennen
- Volltextsuche über Dateiname, Tags, Notizen und Projekte
- Standardprogramm pro Dateityp hinterlegen
- ZIP-Backup der Datenbank
- Optional auch Backup der Originaldateien möglich
- Automatische Updates

## Systemvoraussetzungen

- Windows 10 / 11 (64-bit)
- Internetverbindung für automatische Updates (optional)

## Updates

Installierte Versionen prüfen beim Start automatisch auf Updates und laden neue Versionen im Hintergrund.
